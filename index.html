Documentation

Complete guide to using AptaForge for aptamer sequence generation

Getting Started

AptaForge (Generative Exploration for Aptamer Research) is a web-based platform that uses Monte Carlo Tree Search (MCTS) algorithms to generate optimized aptamer sequences targeting specific proteins.

To begin generating aptamer sequences, navigate to the submission page and configure your job parameters.

Configuration Parameters

Target Configuration

Job Name
A descriptive name for your generation job. This will be used to identify your results when exporting.
Target Protein Sequence
The amino acid sequence of the target protein. The MCTS algorithm will generate aptamer sequences optimized for binding to this protein target.

Generation Parameters

Sequence Length (bp)
The desired length of the generated aptamer sequences in base pairs. Typical range: 10-100 bp. Longer sequences may provide more specificity but take longer to generate.
Number of Candidates
How many candidate sequences to generate simultaneously. Maximum: 10. More candidates provide greater diversity but increase computation time.
MCTS Simulations
The number of Monte Carlo simulations to run per nucleotide position. Higher values improve sequence quality but increase generation time. Range: 1000-10,000.
Exploration Factor
Controls how greedily the selection process is during MCTS. Higher values encourage exploration of less optimal nucleotides, potentially leading to more diverse sequences. Typical values: 0.5-3.
